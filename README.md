# Digi-Products Vault
A high-converting landing page replica for digital products, built with Vite, Vanilla JS, and modern CSS.

## Features
- Mobile-first responsive design.
- Premium dark/orange aesthetic.
- Interactive FAQ accordions.
- Dynamic Vault carousel.
- Glassmorphism effects.

## Setup
1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev` to start the local development server.
